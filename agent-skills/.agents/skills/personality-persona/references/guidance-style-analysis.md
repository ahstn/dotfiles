# Style analysis and classification

<when_to_use>
- Use when asked to analyze writing samples, compare authors, classify tone or register, extract a house style, or convert sample texts into reusable writing guidance.
</when_to_use>

<analysis_method>
- Build a style fingerprint across these axes:
  - rhetorical stance
  - sentence cadence
  - syntactic complexity
  - lexical richness
  - punctuation and typography
  - formatting density
  - evidence posture
  - recurring positive markers
  - recurring negative markers
- Ground every claim in the supplied samples.
- Contrast styles by describing tendencies and distributions, not just labels.
- When the user wants transfer guidance, distill the result into "keep", "avoid", and "optional" rules.
</analysis_method>

<default_output>
1. Fingerprint
2. Contrast
3. Transfer guidance
4. Optional metric bands, if the user wants measurement or eval criteria
</default_output>

<analysis_rules>
- Describe how the writing feels and how it is built.
- Separate voice from channel. A writer can keep the same voice across different publication types while changing structure and formality.
- Note where the style is stable versus where it flexes by audience or medium.
- Prefer vectors and dimensions over named-author mimicry.
</analysis_rules>

<avoid>
- Abstract labels with no evidence.
- A banned-word list as the whole analysis.
- Named-author imitation unless the user supplied the text and explicitly wants near-style transfer.
</avoid>
