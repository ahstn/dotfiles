## Linux dotfiles

Structured to work with GNU Stow
```
$ git clone https://github.com/phazyy/dotfiles.git
$ cd dotfiles/
$ stow terminator
$ stow atom
```
etc, etc..

## Appearance
- GTK: [Arc]
- Shell: [Ciliora-Teria]
- Icons: [La Capitaine]
