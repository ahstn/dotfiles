## Linux dotfiles

Structured to work with GNU Stow
```
$ git clone https://github.com/phazyy/dotfiles.git
$ cd dotfiles/
$ stow terminator
$ stow atom
```
etc, etc..

## Configs
- BSPWM
- Atom Editor
- Fish Shell
- Terminator
- Tmux
- Vim
